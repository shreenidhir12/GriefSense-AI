# GriefSense AI

Predictive Grief Intelligence Platform

## Problem

Millions of people suffer from grief without access to affordable support.

## Solution

GriefSense AI predicts grief stages and provides emotional support while helping therapists monitor high-risk individuals.

## Features

* Grief Stage Detection
* PGD Risk Assessment
* Therapist Dashboard
* AI-Based Support

## Tech Stack

* Python
* Streamlit
* NLP
* Machine Learning

## Future Scope

* Voice Analysis
* Multilingual Support
* Therapist Alerts
* Clinical Validation
